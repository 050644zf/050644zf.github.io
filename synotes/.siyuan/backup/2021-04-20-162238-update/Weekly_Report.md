# 4月16日周报
{: id="20210416194627-683qn4g"}

这周，我们在小组会上讨论了高光谱建模及算法的问题。本周我们着重于噪声建模问题，我们分析了系统各环节可能出现的噪声，并发现噪声更多来自于器件例如光学元件和传感器。
{: id="20210416194627-nuzp09l"}

会后，我学习了有关噪声的相关知识，学习了常见噪声如高斯噪声，椒盐噪声等噪声的特征以及去除方法。
{: id="20210416194627-5xi35ny"}

| 噪声类型 | 分布函数/特征                                 | 场景                      | 去除方法                                    |
| ---------- | ----------------------------------------------- | --------------------------- | --------------------------------------------- |
| 高斯噪声 | 符合均值为 $\bar{z}$ ，方差为 $\sigma$ 的高斯分布 | 传感器热噪声、电路噪声等  | 均值滤波器<br />图像叠加<br />维纳滤波等^1^ |
| 椒盐噪声 | 随机亮点/暗点                                 | 传感器读出错误等          | 中值滤波器等^1^                             |
| 泊松噪声 | 符合泊松分布                                  | 低照度图像、高增益图像^2^ | 自适应中值滤波器等^1^                       |
| 乘性噪声 | 符合瑞利或伽玛分布                            | 相干图像系统，例如激光等  | 同态滤波（乘性→加性)^3^                    |
