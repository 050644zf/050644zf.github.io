# 4月23日周报
{: id="20210417165322-w4mmgmr" updated="20210420155305"}

本周我们在小组会上讨论了关于如何构建稀疏基的问题，会议上主要讨论了论文《Compressive Sensing Spectroscopy Using a Residual Convolutional Neural Network》，对其所使用的残差卷积网络和字典学习方法进行探讨。
{: id="20210420155307-4pm44a8" updated="20210422093342"}

会后，我学习了学习字典的相关知识。
{: id="20210422094246-yq0u0xt" updated="20210422094303"}

给定字典$D=[d_1,d_2,...,d_K]\in \mathbb{R}^{n\times K}$，使得信号$X=[x_1,x_2,...,x_n]\in \mathbb{R}^{n\times N}$可被表示为
{: id="20210422094830-fq0sryy" updated="20210422095911"}

$$
X\approx DA
$$
{: id="20210422095911-7r8wsks" updated="20210422095913"}

其中$A=[\alpha_1,\alpha_2,...,\alpha_N]\in \mathbb{R}^{K\times N}$ 为$X$在$D$下的表示系数，当$\alpha_i$中非零元素远小于$K$时，则称该向量为稀疏的。
{: id="20210422095959-x32terg" updated="20210422100247"}

则字典学习优化问题可表示为：
{: id="20210422100247-nwpcfta" updated="20210422100350"}

$$
\min _{D, A}\left\{\|X-D A\|_{F}^{2}+\lambda_{A} g_{A}(A)\right\}
$$
{: id="20210422100351-1ucolur" updated="20210422100352"}

其中$g_A()$为求稀疏度函数，$\lambda_A$为归一化参数。
{: id="20210422100404-mmy01xm" updated="20210422100455"}

字典学习算法总结：
{: id="20210422101121-6dosrji" updated="20210422101139"}

| 算法      | 主要思想                               |
| ----------- | ---------------------------------------- |
| Sparsenet | 最大似然估计                           |
| MOD       | $l_o$范数衡量稀疏性<br />最小均方误差    |
| K-SVD     | 正交匹配追踪(OMP)<br />奇异值分解(SVD) |
| ODL       | 每次迭代仅处理一个样本                 |
{: id="20210422100456-0zct308" updated="20210422100543"}

# 4月16日周报
{: id="20210416194627-683qn4g"}

这周，我们在小组会上讨论了高光谱建模及算法的问题。本周我们着重于噪声建模问题，我们分析了系统各环节可能出现的噪声，并发现噪声更多来自于器件例如光学元件和传感器。
{: id="20210416194627-nuzp09l"}

会后，我学习了有关噪声的相关知识，学习了常见噪声如高斯噪声，椒盐噪声等噪声的特征以及去除方法。
{: id="20210416194627-5xi35ny"}

| 噪声类型 | 分布函数/特征                                 | 场景                      | 去除方法                                    |
| ---------- | ----------------------------------------------- | --------------------------- | --------------------------------------------- |
| 高斯噪声 | 符合均值为 $\bar{z}$ ，方差为 $\sigma$ 的高斯分布 | 传感器热噪声、电路噪声等  | 均值滤波器<br />图像叠加<br />维纳滤波等^1^ |
| 椒盐噪声 | 随机亮点/暗点                                 | 传感器读出错误等          | 中值滤波器等^1^                             |
| 泊松噪声 | 符合泊松分布                                  | 低照度图像、高增益图像^2^ | 自适应中值滤波器等^1^                       |
| 乘性噪声 | 符合瑞利或伽玛分布                            | 相干图像系统，例如激光等  | 同态滤波（乘性→加性)^3^                    |
{: id="20210416204446-tkh7sks"}
